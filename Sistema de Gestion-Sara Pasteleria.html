⚙️ Supabase: ¿Cómo obtener estos datos?
1. Andá a supabase.com → creá cuenta gratuita → New Project.
2. En tu proyecto: Settings → API.
3. Project URL: copialo tal cual (ej: https://abcd.supabase.co).
4. anon public key: la clave larga que empieza con eyJ....
5. Ejecutá el archivo supabase-setup.sql en SQL Editor → New query → Run.
6. Volvé acá y hacé clic en Conectar.
Sin conexión — configurá Supabase arriba
